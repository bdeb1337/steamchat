## [0.2.1] - 2025-12-22
- Updated dependencies.
- Fixed tray icons so status is represented correctly.
- Signed the macOS build.

## [0.2.0] - 2025-11-26
- Refactored status handling, Steam API handling, and link navigation.
- Improved error handling and disconnected-state behavior.
- Added cleanup for tray and interval handling.
- Updated build and dependency tooling.
- Dropped 32-bit Windows support and added arm64 Windows/Linux builds.

## [0.1.3] - 2024-10-13
- Added extended macOS build info.
- Included the self-signing information needed to resolve macOS permissions issues.

## [0.1.2] - 2024-05-07
- Switched macOS arch-specific builds to universal builds for better Homebrew support.

## [0.1.1] - 2024-05-05
- Improved link handling, tray behavior, and notification click handling.
- Added fixes for several window and connectivity issues.
- Updated Electron packaging dependencies.

## [0.0.9] - 2024-03-24
- Updated Electron to 29.1.5.
- Added early fixes for notification and link handling.

## [0.0.8] - 2024-02-27
- Improved tray behavior and shutdown handling.
- Added auto-launch support and other stability fixes.
- Updated Electron and dependency packaging.
